# Portfolio Islena Polo - React + TypeScript

Portfolio profesional en React, TypeScript, Vite y Bootstrap con estética morada, animaciones, partículas y componentes reutilizables.

## Instalar

```bash
npm install
npm run dev
```

Después abre la URL que te muestre Vite, normalmente `http://localhost:5173`.

## Build

```bash
npm run build
npm run preview
```

## Estructura

- `src/components`: secciones y tarjetas reutilizables.
- `src/hooks`: hooks personalizados para glow del ratón y progreso de scroll.
- `src/data`: proyectos y stack técnico.
- `src/styles`: estilos personalizados sobre Bootstrap.
